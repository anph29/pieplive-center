# pieplive-center

checkout:

```
git clone https://github.com/anph29/pieplive-center-v1.git
```
update
```
git pull
```
commit
```
git add * | <filename> | <folder name>
git commit -m 'message'
git push
```
#REQUIRE:
python >= 3

install project:
``` bash
pip3 install virtualenv
virtualenv pieplive-center
cd pieplive-center
source bin/activate
pip install -r requirements.txt
```

run:
```
python piepme.py
```
